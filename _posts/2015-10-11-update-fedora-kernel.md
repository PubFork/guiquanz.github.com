## Mac OS X 玩 Linux

玩了 5 年多的 Linux 本本，终于还是被我淘汰了，现在落寞到在 Mac OS X 上用虚拟机继续 Linux 开发了。当然，主要还用 Mac 开发 APP 还有其他类型的应用。Fedora 20 的内核版本还是有些老，很多大杀器都无法使用，只有升级版本了。终于，也步入 Kernel 4.1 的时代。


## 升级操作

升级操作很简单，升级后可能会碰上各种坑，所以请慎重。

```bash

$ sudo yum update

# 或

$ sudo yum update kernel*

```
